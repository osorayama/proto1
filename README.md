# ①課題名
海外滞在者向け生活圏マップ型コミュニティ（Community Map MVP）


## ②課題内容（どんな作品か）
- 海外中長期滞在中の日本人向けの位置情報ベースのコミュニティプラットフォーム
- 地図上で近くの投稿（売買・譲渡、ルームメイト、求人、イベント）を閲覧できる
- 投稿者とチャットでやり取りができ、ブロック・通報機能で安全性を確保
- スマホファーストのレスポンシブデザインで、PWA対応

## ③アプリのデプロイURL
https://sottra.sakura.ne.jp/proto1/public/register.php


## ④アプリのログイン用IDまたはPassword（ある場合）
- 新規登録してご利用ください

## ⑤工夫した点・こだわった点
- 位置情報を活用した半径フィルタリングで、近くの投稿のみを表示
- チャット機能は5秒ポーリングでリアルタイム風の体験を実現（WebSocket不要）
- 水色を基調とした清潔感のあるUI設計で、直感的な操作性を重視
- iPhone SE等の小型デバイスでも快適に使えるよう、下部タブバーとの重なりを調整


## ⑥難しかった点・次回トライしたいこと（又は機能）
- APIファイルの依存関係整理（includes/helpers.phpに集約）
- チャット画面のレイアウト調整（固定ヘッダー・入力エリアとスクロール領域の分離）
- 次回：画像アップロード機能、WebSocketによるリアルタイムチャット、プッシュ通知の実装


## ⑦フリー項目（感想、シェアしたいこと等なんでも）
- PHP + MySQLのシンプルなスタックでも、位置情報とチャット機能を組み合わせた実用的なアプリが構築できることを実感しました
- さくらサーバーへのデプロイ時に環境差異による問題が多発しましたが、config.phpによる環境自動判定で解決できました
- 今後は画像投稿やOAuth認証、管理画面などを追加して、実際のサービスとして運用できるレベルまで機能拡充していきたいです
